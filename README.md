# Kuva

#### .NET CLI

Instalación:

```sh
dotnet tool install -g
```

Versión:

```sh
dotnet --version
```

Instalar template local:

```sh
dotnet new install .
```

Ver template local:

```sh
dotnet new list
```

Desinstalar template local:

```sh
dotnet new uninstall
```

Crear nuevo proyecto basado en el template:

```sh
dotnet new webapicustomtemplate -o "NombrePersonalizado"
```
